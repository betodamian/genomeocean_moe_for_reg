# Curated RBS / Shine-Dalgarno Database

**Created:** 2026-06-24  
**Status:** Sources verified; schema defined; raw data to be fetched from accessions below (Week 1 task).

No pre-built multi-organism experimental RBS database exists. This directory is a curated
compilation from ribosome-profiling, dRNA-seq, and biochemical studies that provide
genome-wide, experimentally confirmed translation initiation site (TIS) classifications
across the five core genomes in this project.

---

## Files

| File | Contents |
|---|---|
| `sources.tsv` | One row per study — organism, method, TIS counts, SD/leaderless breakdown, data accession, PMID |
| `schema.tsv` | Header-only TSV defining the per-entry columns for the full database |
| `README.md` | This file — evidence hierarchy, curation rules, fetch instructions |

The full per-entry database (`rbs_entries.tsv`) is generated by running `fetch_and_build.py`
against the accessions in `sources.tsv`. It is not committed (large, regenerable).

## Evidence hierarchy

Only the following evidence types qualify as ground truth (consistent with §12 of the
research plan — no software-predicted labels):

| Tier | Evidence type | SD-class assigned | Used as |
|---|---|---|---|
| **T1 (strongest)** | TIS-arrest ribosome profiling (Ribo-RET / TetRP) | SD / UNSD / leaderless | Positive and negative |
| **T1** | Standard Ribo-seq with ribosome-signature TIS calling | SD / UNSD / leaderless | Positive and negative |
| **T2** | dRNA-seq TSS where TSS overlaps annotated start codon | leaderless (no 5′ UTR / no SD) | Leaderless negative class |
| **T2** | Biochemically confirmed SD interaction (anti-SD pairing, ΔG measurement) | SD | Positive (small curated sets) |
| **Excluded** | Any computational / predicted label (RBS Calculator, Prodigal, etc.) | — | Never used as ground truth |

SD-class definitions:
- **SD**: 5′ UTR present; GGAGG-like motif detectable at −20..−1 (canonical or extended);
  ΔG pairing to anti-SD ≤ −3.4 kcal/mol (Salis RBS Calculator threshold, used for
  classification only, not as the evidence source).
- **UNSD (leadered non-SD)**: 5′ UTR present (≥1 nt); no detectable SD motif.
- **Leaderless**: TSS coincides with start codon (0–2 nt 5′ UTR); no SD possible.

## Per-organism sources

### E. coli K-12 (genome: GCF_000005845.2 / NC_000913.3)

Three independent TIS datasets — use the intersection of ≥2 for high-confidence labels:

1. **Nakahigashi et al. 2016 (TetRP)** — PMID 27013550  
   Accession: PRJDB2960 (DDBJ)  
   ~360 novel intergenic TIS; ~150 reannotated known-gene starts; E. coli BW25113.

2. **Meydan et al. 2019 (Ribo-RET)** — PMID 30904393  
   Accession: GSE122129 (GEO)  
   991 primary TIS (BW25113), 1,990 (BL21); 239 internal TIS candidates.  
   ~54% of expressed genes carry SD motif.

3. **Saito et al. 2020 (ribosome profiling, ΔaSD mutant)** — PMID 32424076  
   Accession: GSE135906 (GEO)  
   2,217 genes analyzed; confirms ~54% SD usage; SD affects efficiency, not site selection.

**Leaderless negative class for E. coli:** E. coli has very few leaderless genes (~1%).
Use UNSD-leadered genes (non-SD 5′ UTR present) as the Tier-2 negative instead,
drawn from Meydan et al. 2019 supplementary table.

---

### M. tuberculosis H37Rv (genome: GCF_000195955.2 / NC_000962.3)

Two complementary datasets — the best single-organism leaderless negative source:

1. **Zhu et al. 2021 (Ribo-seq)** — PMID 33513356  
   Accession: E-MTAB-8835 (ArrayExpress)  
   3,569 genes quantified (88.5% of translatome).  
   SD: 546 (15%) | UNSD: 678 (19%) | Leaderless: 497 (14%) | Operons/unassigned: ~51%

2. **Cortes et al. 2013 (dRNA-seq)** — PMID 24268774  
   Accession: SRP028740 (SRA)  
   4,006 TSS mapped; 26% of TSS-mapped genes are leaderless (TSS = start codon).

**Why MTB is the primary leaderless negative source:** 497 ribosome-profiling-confirmed
leaderless genes + 1,040 dRNA-seq-confirmed leaderless transcripts provide the largest
same-position, no-SD negative class of any organism in this panel.

---

### B. subtilis 168 (genome: GCF_000009045.1 / NC_000964.3)

1. **Lalanne et al. 2017 (Ribo-seq)** — PMID 29144454  
   Accession: GSE95211 (GEO)  
   Genome-wide translation stoichiometry; ~78% of B. subtilis genes carry SD motif
   (consistent with genomic estimates; Wen et al. 1995).

2. **Bhatt et al. 2024 (Ribo-seq, sporulation time course)** — PMID 39179838  
   Accession: GSE249450 (GEO)  
   Comprehensive multi-stage dataset; use for additional TIS calls.

**Notes:** B. subtilis has the highest known SD usage of any model organism (~78%).
Very few leaderless genes. Use as a high-SD positive-enriched organism.

---

### S. aureus NCTC 8325 (genome: GCF_000013425.1 / NC_007795.1)

1. **Kohl et al. 2026 (Ribo-seq + cryo-EM)** — PMID 41680142  
   Published: Nature Communications, February 2026  
   DOI: 10.1038/s41467-026-69079-8  
   Accession: see paper data availability (GEO/SRA — retrieve from paper Methods).  
   High-resolution TIS mapping; genome-wide SD usage quantified with extended-SD motifs.  
   Key finding: S. aureus relies on extended, start-codon-proximal SD interactions that
   differ mechanistically from E. coli; several native S. aureus TIS are not decoded by
   E. coli ribosomes. Very few leaderless genes.

---

### P. aeruginosa PAO1 (genome: GCF_000006765.1 / NC_002516.2)

No dedicated genome-wide TIS profiling study found as of Jun 2026. Strategy:
- **Positives:** derive from NCBI PGAP annotated CDS starts (5,572 genes); apply SD
  scoring (ΔG pairing to anti-SD 5′-CCUCCUUA-3′) to classify SD / UNSD.
- **Labels:** annotated-start derived (Tier-2 in evidence hierarchy — weaker).
- **Action:** search for Pseudomonas Ribo-seq datasets on GEO before Week 1 ends;
  if found, upgrade to T1 evidence. Otherwise use PGAP-derived labels and flag as
  lower-confidence in per-entry `notes` column.

## Schema column definitions

| Column | Type | Description |
|---|---|---|
| `entry_id` | string | Unique ID: `{source_id}_{locus_tag}` |
| `organism` | string | Binomial species name |
| `strain` | string | Strain designation |
| `genome_accession` | string | NCBI RefSeq assembly accession |
| `locus_tag` | string | NCBI locus tag (e.g. b0001) |
| `gene_name` | string | Gene name if known, else NA |
| `start_coord` | int | Genomic start coordinate of CDS (1-based) |
| `end_coord` | int | Genomic end coordinate of CDS |
| `strand` | +/- | Strand |
| `sd_window_seq` | string | 20 nt genomic sequence ending at start codon −1 (i.e. positions −20..−1 relative to start A of ATG) |
| `sd_window_start` | int | Genomic start of the SD window |
| `sd_window_end` | int | Genomic end of the SD window |
| `sd_class` | SD / UNSD / leaderless | Classification (see evidence hierarchy above) |
| `sd_motif` | string | Best SD motif found (GGAGG / GAGG / AGG / AAGG / extended, or NA) |
| `sd_motif_start_rel` | int | Position of SD motif start relative to ATG (negative = upstream; e.g. −10) |
| `free_energy_kcal_mol` | float | ΔG of SD:anti-SD pairing (Salis RBS Calculator, for classification only); NA if leaderless |
| `evidence_type` | string | T1_riboret / T1_tetrp / T1_riboseq_TIS / T2_drnaseq_leaderless / T2_pgap_derived |
| `source_id` | string | Foreign key to `sources.tsv` source_id |
| `PMID` | int | PubMed ID of the source paper |
| `notes` | string | Free text — any caveats, strain differences, conflicting calls |

## Curation rules

1. **One entry per experimentally confirmed TIS per organism.** Where multiple studies
   call the same start (same locus_tag + start_coord), keep the highest-tier evidence
   and record the other studies in `notes`.

2. **SD classification uses the sd_window_seq + ΔG rule, not predicted labels.**
   Compute ΔG using the Salis RBS Calculator v2.1 Python library (offline, no API call).
   Threshold: ΔG ≤ −3.4 kcal/mol → SD; ΔG > −3.4 and 5′ UTR ≥ 1 nt → UNSD;
   5′ UTR = 0 nt → leaderless.

3. **Leaderless calls require TSS evidence.** Do not call leaderless from start coordinates
   alone — require a dRNA-seq or TIS-profiling dataset confirming the TSS overlaps the
   start codon. PGAP-derived Tier-2 starts do not get a leaderless label.

4. **SD class for P. aeruginosa is flagged T2_pgap_derived until Ribo-seq data is found.**
   Do not include P. aeruginosa entries in the headline Tier-2 negative class until
   upgraded to T1 evidence.

5. **Conflicting calls (e.g. two studies disagree on start site):** keep both entries,
   add `conflict=true` in `notes`, and exclude from the 80/20 split's training set
   (move to a held-aside conflict set).
